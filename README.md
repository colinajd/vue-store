# Vue Store
A responsive web-store boilerplate created in Vue2

### Example
A running version of this can be found at: https://vue-store.herokuapp.com/

To start:

```bash
$ yarn
```

To develop:

```bash
$ yarn run dev
```

To build for production:

```bash
$ yarn run build
```

To run in HTTP cart-container

```bash
$ yarn start
```
